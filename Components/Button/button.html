Button
Buttons are the soul of an interactive Web Application. Buttons allow users to take actions, and make choices, with a single tap.
Solid Button
Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app. 
                                      
                        <button class="btn">DEFAULT</button>
                        <button class="btn btn_Primary">PRIMARY</button>
                        <button class="btn btn_Secondary">SECONDARY</button>
                        <button class="btn btn_Error">ERROR</button>
                        <button class="btn btn_Success">SUCCESS</button>
Link Button
Want to attach a link action, at a button tap? Use Link Button 
                    
                
                <button class="btn  outline_C"><a href="../Avatar/avatar.html">AVATAR</button>
                <button class="btn  outline_A"><a href="../Alert/alert.html"ALERT></button>
                <button class="btn  outline_B"><a href="../Badges/badges.html"BADGES></button>
Icon Button
The icon button component has different styles, each with its own semantic meaning. To use the button use .btn .btn-icon 
                        
                            <button class="btn btn_Icon">
                                <span class="material-icons">add_shopping_cart</span>
                            Add to cart</button>
                           <button class="btn btn_Icon">
                               <span class="material-icons">delete_outline</span>
                           Delete</button>
Floating Button
The button component has different styles, each with its own semantic meaning. To use the button using floating-btn. 
                        
                            <button class="btn btn_Floating">
                                <span class="material-icons">contact_phone</span>
                            </button>
                           <button class="btn btn_Floating">
                               <span class="material-icons">comment</span>
                           </button>
