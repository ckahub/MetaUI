Avatar
Avatars are user profile pictures, usually found in circular or sqaure shapes. Avatars can be used for 'container' objects e-commerce,social media, and many other apps â€” to give them personalize approach tom user.
Profile Avatar
Profile Avatars can be created using the class 'avatar', inside the img tag. 
                
                    <img class="avatar" src="../../assests/Avatar1.jpg" alt="avatar1" >
                    <img class="avatar" src="https://unsplash.com/photos/jzz_3jWMzHA" alt="avatar1" >
Avatar Sizes
Size of the avatars can be changed by adding sm | md | lg class to the existing avatar class. 
                
                    Using the sm | md | lg class names

                     <img class="avatar sm" src="https://url-of-the-image" alt="Avatar0">
                     <img class="avatar lg" src="https://url-of-the-image" alt="Avatar1">
                     <img class="avatar md" src="https://url-of-the-image">
Square Avatar
Adding class 'square' to existing class will generate sqaure shaped avatar.
                
                    Using the square class 
                     <img class="avatar square sm" src="https://url-of-the-image" alt="Avatar0">
                     <img class="avatar square md" src="https://url-of-the-image" alt="Avatar1">
                     <img class="avatar square lg" src="https://url-of-the-image" alt="Avatar5">
