Input
The input field can be specified using where a user can enter data. The input tag is used within form element to declare input controls that allow users to input data. An input field can be of various types depending upon the attribute type. The Input tag is an empty element which only contains attributes.
Text Box
This is a simple input field feild to enter text. 
                                      
                        <label for="input">Name</label>
                        <input class="input" type="text" placeholder="Enter ur name?">
Validation
This is a multiple input feild that response with variety. It has three types of validation response. 
                                      
                        <input class="input input_Success" type="text" placeholder="Validation Successful">
                        <input class="input input_Warn" type="text" placeholder="Validation Warning">
                        <input class="input input_Error" type="text" placeholder="Validation Error">
