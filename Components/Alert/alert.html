Alert
An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.
Simple Alert
Simple Aletrs can be created by using alertclass ,followed by variant name.
Primary!This an Primary Alert! check it out
Error!!This an Error Alert! check it out
Success!This an Success Alert! check it out
Warning!This an Warning Alert! check it out 
                    
                        <div class="alert primary">
                            <strong> Primary! </strong>This is an primary Alert! check it out.
                        </div">
                        <div class="alert error">
                            <strong> Error! </strong>This is an error Alert! check it out.
                        </div">
                        <div class="alert success">
                            <strong> success! </strong>This is an success Alert! check it out.
                        </div">
                        <div class="alert warning">
                            <strong> Warning! </strong>This is an warning Alert! check it out.  
                        </div">
Outline Alert
Outline alerts are created by using alert class, followed by outline-variant.
Primary!This an Primary Alert! check it out
Error!!This an Error Alert! check it out
Success!This an Success Alert! check it out
Warning!This an Warning Alert! check it out 
                                    
                      <div class="alert outline_Primary">
                         <strong>Primary!</strong>This is an Primary Alert! check it out.
                      </div>
                      <div class="alert outline_Error">
                         <strong>Error!</strong> This is an Error Alert! check it out.
                      </div>
                    <div class="alert outline_success">
                         <strong>Success!</strong>This is an Success Alert! check it out. 
                     </div>
                     <div class="alert outline_warning">
                         <strong>Warning!</strong>This is an Warning Alert! check it out.
                     </div>
Alert With Icons
Alert with Icons can be created by using Material Icons using Icon and alertclass ,followed by variant name.
fingerprint Primary!This an Primary Alert! check it out
error_outline Error!!This an Error Alert! check it out
done_outline Success!This an Success Alert! check it out
warning_amber Warning!This an Warning Alert! check it out 
                    
                        <div class="alert primary">                        
                            <span class="material-icons">fingerprint</span>
                                <strong> Primary! </strong>This is an primary Alert! check it out.
                        </div>
                        <div class="alert error">
                            <span class="material-icons">error_outline</span>    
                                <strong> Error! </strong>This is an error Alert! check it out.
                        </div>
                        <div class="alert success">
                            <span class="material-icons">done_outline</span> 
                                <strong> success! </strong>This is an success Alert! check it out.
                        </div>
                        <div class="alert warning">
                            <span class="material-icons">warning-amber</span>
                                <strong> Warning! </strong>This is an warning Alert! check it out. 
                        </div>
